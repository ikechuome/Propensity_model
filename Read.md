# Propensity to Buy Model

## Project Structure
- `src/` — core Python scripts
- `notebooks/` — training and testing notebooks
- `data/` — input datasets
- `models/` — saved model pickle files
- `outputs/` — prediction results

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run training: open `notebooks/Model_demo.ipynb`
3. Run predictions: open `notebooks/Test_model.ipynb`
